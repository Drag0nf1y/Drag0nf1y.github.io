MYSQL 8.0+Navicat Premium 15联动

环境：windows 10

1 mysql 安装

1.下载

需要注册账号，在官网下载安装包：https://dev.mysql.com/downloads/file/?id=499256

下载后解压到放软件的目录下：

2 my.ini

原本是没有my.ini这个文件，在下面的步骤需要将这个文件加进去。

根据自己的需求，将安装目录和数据存放目录进行修改。

3 配置环境变量

在系统变量->Path->添加值

4 管理员方式运行cmd
切换到刚刚的安装目录：
mysqld –initialize –console

此时会出现一个密码

后面跟的就是初始密码需要记住，以便在后面修改密码。

出错

在mysql的bin目录下面执行： mysqld –install

报错：

Install/Remove of the Service Denied

解决办法：

打开cmd.exe程序的时候选择“用管理员身份打开”。

5 安装服务

mysqld –install

默认为mysql的数据库。

net start mysql

可以看见启动了服务

6 修改密码

mysql -u root -p

进入mysql的控制界面

ALTER USER ‘root’@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘新密码’;

输入之前记住的密码，再重置密码即可。

2 基本操作

每条命令之后，必须加上分号这是mysql命令特点

进入mysql: mysql -u root

退出mysql: exit

创建数据库: CREATE DATABASE test;

查看数据库： SHOW DATABASES;

访问数据库：USE　test;

查看表： SHOW TABLES;

导入表到数据库：mysql -D test -u root -p < D:\\My\\groupfiles\\finish.sql

查看表头：

DESCRIBE finish;

3 NAVICAT PREMIUM 15

下载地址：http://www.navicat.com.cn/download/navicat-premium

仅有15天的试用期，可以破解。

连接后，可以通过这个Navicat来可视化和操作数据库中的内容。

之后每次开始协同操作，需要：

先以管理员的方式运行cmd: net start mysql

然后打开这个Navicat来可视化和操作数据库中的内容。
